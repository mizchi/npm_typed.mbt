# mizchi/npm_typed/classnames

MoonBit FFI bindings for [classnames](https://www.npmjs.com/package/classnames).

## Installation

```bash
moon add mizchi/js
moon add mizchi/npm_typed
npm install classnames
```

Add to your `moon.pkg.json`:

```json
{
  "import": [
    "mizchi/js",
    "mizchi/js/core",
    "mizchi/npm_typed/classnames"
  ]
}
```
