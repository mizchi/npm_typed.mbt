# mizchi/npm_typed/zustand

MoonBit FFI bindings for [zustand](https://www.npmjs.com/package/zustand).

## Installation

```bash
moon add mizchi/js
moon add mizchi/npm_typed
npm install zustand
```

Add to your `moon.pkg.json`:

```json
{
  "import": [
    "mizchi/js",
    "mizchi/js/core",
    "mizchi/npm_typed/zustand"
  ]
}
```
