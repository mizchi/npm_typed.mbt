# mizchi/npm_typed/lit_html

MoonBit FFI bindings for [lit_html](https://www.npmjs.com/package/lit_html).

## Installation

```bash
moon add mizchi/js
moon add mizchi/npm_typed
npm install lit_html
```

Add to your `moon.pkg.json`:

```json
{
  "import": [
    "mizchi/js",
    "mizchi/js/core",
    "mizchi/npm_typed/lit_html"
  ]
}
```
