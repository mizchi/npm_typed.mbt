# mizchi/npm_typed/pptr_testing_library

MoonBit FFI bindings for [pptr-testing-library](https://www.npmjs.com/package/pptr-testing-library).

## Installation

```bash
moon add mizchi/js
moon add mizchi/npm_typed
npm install pptr-testing-library
```

Add to your `moon.pkg.json`:

```json
{
  "import": [
    "mizchi/js",
    "mizchi/js/core",
    "mizchi/npm_typed/pptr_testing_library"
  ]
}
```
