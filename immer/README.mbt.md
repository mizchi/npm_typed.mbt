# mizchi/npm_typed/immer

MoonBit FFI bindings for [immer](https://www.npmjs.com/package/immer).

## Installation

```bash
moon add mizchi/js
moon add mizchi/npm_typed
npm install immer
```

Add to your `moon.pkg.json`:

```json
{
  "import": [
    "mizchi/js",
    "mizchi/js/core",
    "mizchi/npm_typed/immer"
  ]
}
```
