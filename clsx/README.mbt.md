# mizchi/npm_typed/clsx

MoonBit FFI bindings for [clsx](https://www.npmjs.com/package/clsx).

## Installation

```bash
moon add mizchi/js
moon add mizchi/npm_typed
npm install clsx
```

Add to your `moon.pkg.json`:

```json
{
  "import": [
    "mizchi/js",
    "mizchi/js/core",
    "mizchi/npm_typed/clsx"
  ]
}
```
