# mizchi/npm_typed/valibot

MoonBit FFI bindings for [valibot](https://www.npmjs.com/package/valibot).

## Installation

```bash
moon add mizchi/js
moon add mizchi/npm_typed
npm install valibot
```

Add to your `moon.pkg.json`:

```json
{
  "import": [
    "mizchi/js",
    "mizchi/js/core",
    "mizchi/npm_typed/valibot"
  ]
}
```
