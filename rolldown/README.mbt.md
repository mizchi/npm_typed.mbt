# mizchi/npm_typed/rolldown

MoonBit FFI bindings for [rolldown](https://www.npmjs.com/package/rolldown).

## Installation

```bash
moon add mizchi/js
moon add mizchi/npm_typed
npm install rolldown
```

Add to your `moon.pkg.json`:

```json
{
  "import": [
    "mizchi/js",
    "mizchi/js/core",
    "mizchi/npm_typed/rolldown"
  ]
}
```
